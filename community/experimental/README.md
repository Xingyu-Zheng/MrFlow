# Experimental Contributions

This directory collects early community contributions, prototypes, and workflows that are useful to share before broader testing.

Community contributions that have been tried by the maintainers and are ready for broader reuse may later be moved to `community/verified/`, or promoted into the main `examples/` or `ComfyUI-MrFlow/` folders when they become part of the official workflow.

## Pending Verification

- [MrFlow Fast Diffusion Space](https://huggingface.co/spaces/hugging-apps/mrflow-fast-diffusion): an interactive Hugging Face Space built by the Hugging Face open-source team. The current Space is useful for public testing, but still appears to have a dull/grayish color issue from Stage 2 onward that needs to be fixed.
- [Rebels_MrFlow](https://github.com/RealRebelAI/Rebels_MrFlow): a community ComfyUI extension and workflows exploring MrFlow-style staged sampling for Krea-2 and Z-Image Turbo.
